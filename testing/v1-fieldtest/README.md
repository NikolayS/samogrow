## Diagnosis (2026-07-13)

| Photo | What it shows |
|---|---|
| `img/2026-07-13-parsley-decline.jpeg` | Parsley with yellowing/browning leaf margins — the decline that kicked off the investigation. |
| `img/2026-07-13-mint-necrotic.jpeg` | Spearmint with brown necrotic leaf patches. |
| `img/2026-07-13-reservoir-sediment.jpeg` | Sediment on the reservoir floor — sparked the "undissolved nutrient vs. residual soil" debate. |
| `img/2026-07-13-reservoir-128x10.jpeg` | TDS-3 in the reservoir reading **128 with ×10 lit = 1,280 ppm**. The ×10 flag is the whole story. |
| `img/2026-07-13-ph-strip.jpeg` | pH strip reading ~6.0 (yellow) — in target band. |
| `img/2026-07-13-roots-netcup.jpeg` | Net cup lifted out: roots pale/cream and firm (no rot), short/still developing. |

**One-line lesson:** cheap TDS meters (HM Digital TDS-3) display a **×10 multiplier**; missing it made 1,300–2,000 ppm read as "130–200", so the solution was ~2× too strong and the leaves salt-burned. Target = displayed **80–110 with ×10 lit** = 800–1,100 ppm.

*Photos are cropped to the subject. A plain tap-water meter check was also done (meter read a sane value → confirmed working) but isn't included here.*
