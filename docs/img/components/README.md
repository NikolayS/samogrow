# Component photos

These are **vendor product photos** collected for the samogrow personal DIY
garden guide. Each image is the main product shot from the retailer (mostly
Amazon; the storage tote is The Home Depot's HDX product photo) and is used
here purely for illustration alongside the shopping list.

- Each file maps to a shopping-list item; see `manifest.json` for the
  `file` → `itemName` → `sourceUrl` mapping and the original `imageUrl`.
- Images were downloaded from the vendor page and normalized with macOS `sips`
  to max 480px wide, JPEG quality ~75, to keep the folder small.
- Product/brand images remain the property of their respective owners. If you
  reuse or publish this guide, verify the current product and swap in your own
  photos as needed.

Source pages are linked from each item in `spec/SHOPPING-LIST.md`.
